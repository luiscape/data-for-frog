Network Mapping of Indicators

In this experiment we are using the [new CHD coding scheme](https://github.com/luiscape/data-for-frog/blob/gh-pages/chd_coding_scheme.pdf) to create a network map that represents the relationship among indicators. Each color represents a "level" in the CHD coding scheme.

The levels follow this order: List (Global or Local) >> Class (Operational or Baseline) >> Topic (e.g. Health, Logistics) >> Indicator code
